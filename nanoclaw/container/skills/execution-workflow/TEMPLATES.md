## Debug Checklist Template

```markdown
# Debug Checklist

## 1. Reproduce
- [ ] Exact steps documented
- [ ] Input, environment, and expected/actual behavior captured

## 2. Scope
- [ ] Affected modules identified
- [ ] Regression window estimated

## 3. Signals
- [ ] Logs collected
- [ ] Error traces collected
- [ ] Metrics/telemetry reviewed

## 4. Hypotheses
- [ ] Top 3 hypotheses listed
- [ ] Fastest falsification test defined for each

## 5. Fix
- [ ] Minimal fix applied
- [ ] Side effects reviewed

## 6. Verify
- [ ] Targeted test added/updated
- [ ] Related tests passed
- [ ] Manual verification done

## 7. Close
- [ ] Root cause documented
- [ ] Preventive action recorded
```
